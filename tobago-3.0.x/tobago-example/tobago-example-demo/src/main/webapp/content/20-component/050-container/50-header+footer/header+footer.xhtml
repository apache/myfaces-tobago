A header / footer can be set via the <tc:header/> respectively <tc:footer/> tag.

In this simple example, a <tc:object/> show a page with a fixed header and footer. Also you can access this site in a .

<tc:header fixed="true">
  <tc:style customClass="bg-inverse"/>
    <p>This is the header.</p>
  </tc:header>
  <p>Content between header and footer.</p>
  ...
  <tc:footer fixed="true">
    <p>This is the footer.</p>
</tc:footer>
