The flow layout can be used with the <tc:flowLayout/> tag. By default, the content is displayed in a flow layout.

The buttons are displayed successively at the same line. Components which doesn't fit in the current line are moved to the next one. Components like an outputfield or a checkbox are displayed in a separate line.

 
<tc:flowLayout textAlign="left">
 
<tc:flowLayout textAlign="right">
 
<tc:flowLayout textAlign="center">
 
<tc:flowLayout textAlign="justify">
