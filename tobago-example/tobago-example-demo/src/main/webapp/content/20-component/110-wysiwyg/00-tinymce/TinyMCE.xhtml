This demo uses the CDN of TinyMCE. For more information to setup see .

The setup needs an entry in tobago-config.xml, because of CSP.

This page contains two <tc:script/> tags, one for the TinyMCE and the other for the integration in this application. The <tc:textarea/> contain <tc:dataAttribute name="html-editor" value="tinymce"/>, to mark this textarea to be used as rich text editor. The initialization is initialted by the tinymce.js script.
