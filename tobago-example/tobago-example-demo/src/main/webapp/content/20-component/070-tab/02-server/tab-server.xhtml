Tabs can be created with one or more <tc:tab/> within a <tc:tabGroup>. Within a tab, content can be added like in a section or a panel.

Changing tabs is client sided by default. To change them server sided, the switchType attribute must have the value "reloadPage". All tabgroups on this page are server sided.
