Create a figure with <tc:figure/> around an image, video or something else. The label can be set via the label attribute or with the label facet.

For simple captions use the label attribute.

<tc:figure label="Picture of the alps."> <tc:image value="image/alps.png"/> </tc:figure>

For more complex captions use <f:facet name="label">. In this example the word 'tobago' is colored red.

The tobago logo.
