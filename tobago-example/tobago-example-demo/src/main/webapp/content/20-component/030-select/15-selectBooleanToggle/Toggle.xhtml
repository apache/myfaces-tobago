The tag <tc:selectBooleanToggle/> generate a boolean toggle.

<tc:selectBooleanToggle label="Toggle" value="true"/>

In this example, the selected items are displayed in an output field after the submit-button is pressed.

<tc:selectBooleanToggle label="A" value="\#{selectBooleanToggleController.a}"/>
<tc:out label="Selected items: " value="\#{selectBooleanToggleController.selectedItems}"/>

The output fields always shows the current value of the toggle. With <f:ajax render="output"/>, the output field will be rerendered, after the value changed in the <tc:selectBooleanToggle label="Character" value="#{selectBooleanToggleController.d}"/>. The ID of the output field and the value in the render attribute of the ajax tag has to be the same.
