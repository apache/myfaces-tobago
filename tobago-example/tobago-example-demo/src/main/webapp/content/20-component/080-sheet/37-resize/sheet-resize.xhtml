You can set the columns to resizable with setting the columns attribute. It contains a space separated list of layout tokens '<n>fr', '<x>px', '<x>%' or 'auto', where x is a number and n is an integer. To have a full automatic layout, just leaf the columns attribute out like in the last example on this page. Here we left 5% space for the possible vertical scrollbar. The sum is not needed to be 100%. Ist possible to have a lower or a higher sum. Here you have a horizontal scroll bar, because the sum of the widths is 130% + 1000px. No columns attribute is used here. The columns are automatically layed out, and aren't resizeable.
