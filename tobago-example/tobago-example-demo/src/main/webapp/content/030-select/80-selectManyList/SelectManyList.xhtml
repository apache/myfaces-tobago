The <tc:selectManyList/> component has a filter attribute with four predefined values: contains, startsWith, containsExact, startsWithExact

Choose between different filter types.

Dependent of the type of the filter: Ignores case. Ignores acute (é), grave (è), circumflex (â, î or ô), tilde (ñ), umlaut and dieresis (ü or ï), and cedilla (ç).

Server-side filtering is enabled by wrapping the select items within a tc:filter component. Footer text is defined using the footer attribute. For advanced customization, there is also a footer facet.

<tc:selectManyList value="\#{selectManyListController.selectedSolarObjects}" footer="\#{selectManyListController.footerText}"> <tc:filter query="\#{selectManyListController.query}" minimumCharacters="2"> <tc:selectItems value="\#{selectManyListController.solarObjects}" var="planet" itemLabel="\#{planet.name}" itemValue="\#{planet}"/> </tc:filter> </tc:selectManyList>
