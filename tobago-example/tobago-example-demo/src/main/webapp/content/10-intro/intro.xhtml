Tobago is a user interface framework for developing webapplications. The technical basis of the framework is JavaServer Faces. The main core represents a theme oriented JSF Render Kit.

Tobago provides all standard controls of HTML like inputs, textareas, radiobuttons, checkboxes and buttons. Additionally Tobago provides you with complex controls, which are not natively supportet by simple HTML, including sheets, trees and tabs.

The main goal of Tobago is the development of applications which are interactive from the users point of view. Tobago makes it easy to validate input and helps the user to enter correct data.

Tobago provides several advanced layout managers to fit for specific use cases by using CSS3 and HTML5. This includes a 12 columns layout from and a flex layout which replace the old grid layout from Tobago 2.

If this is the public demo and you want to build it on your own, read the following instructions.

First of all you need (at least version 3.0.4) and or higher.

After that, check out the source code from subversion repository.
svn checkout http://svn.apache.org/repos/asf/myfaces/tobago/trunk tobago3
Alternatively you can get the source code from the .

Go to the tobago3 directory, and build Tobago with mvn install.

Go to 'tobago3/tobago-example/tobago-example-demo' and start the demo with mvn jetty:run.
