Links can be grouped with the <tc:links/> tag. Just add some <tc:link/> or <tc:links/> tags.

Currently there is no difference between <tc:link/> and <tc:links/>.

Tag Library Documentation: |

A simple example with three links.

<tc:links> <tc:link label="Left"/> <tc:link label="Center"/> <tc:link label="Right"/> </tc:links>

A dropdown menu can also be created.

<tc:links> <tc:link label="Dropdown" omit="true"> <tc:link label="Action 1"/> ... </tc:link> <tc:link label="Center"/> <tc:link label="Right"/> </tc:links>

Vertical link group.

<tc:links orientation="vertical"> <tc:link label="Top"/> <tc:link label="Middle"/> <tc:link label="Bottom"/> </tc:links>

This example shows how to disable all links in a link group. It's also possible to set disable to false e.g. for one link.

<tc:links disabled="true"> <c:link label="Left" disabled="false"/> <tc:link label="Center"/> <tc:link label="Right"/> </tc:links>
